# لوحة تحكم المقالات - Admin Panel Documentation

## نظرة عامة
لقد تم إنشاء لوحة تحكم كاملة لإدارة مقالات المدونة. يمكنك من خلالها إنشاء وتعديل وحذف المقالات بسهولة.

## المميزات الرئيسية

### 1. لوحة التحكم الرئيسية
- إحصائيات مرئية لعدد المقالات
- عدد المقالات المنشورة
- عدد الفئات المتاحة
- إجراءات سريعة للوصول للمقالات

### 2. إدارة المقالات
- **عرض جميع المقالات**: قائمة شاملة بجميع المقالات مع تفاصيل سريعة
- **إنشاء مقالة جديدة**: نموذج متقدم لإنشاء مقالات بسهولة
- **تعديل المقالات**: تحديث أي مقالة موجودة
- **حذف المقالات**: حذف المقالات غير المرغوبة
- **عرض المقالة**: رابط سريع لعرض المقالة على الموقع

## كيفية الوصول

### الرابط الرئيسي
```
https://your-domain.com/admin
```

### الروابط المتاحة
- **لوحة التحكم**: `/admin`
- **المقالات**: `/admin/articles`
- **مقالة جديدة**: `/admin/articles/new`
- **تعديل مقالة**: `/admin/articles/[id]/edit`

## نموذج إنشاء/تعديل المقالة

### المعلومات الأساسية
- **العنوان** (مطلوب): عنوان المقالة
- **الملخص**: ملخص قصير يظهر في قوائم المقالات
- **الفئة** (مطلوب): تصنيف المقالة
- **صورة الغلاف**: رابط الصورة الرئيسية للمقالة أو مسار محلي داخل `public/`

### معلومات SEO
- **عنوان SEO**: العنوان الذي يظهر في محركات البحث (أقصى 60 حرف)
- **وصف SEO**: الوصف الذي يظهر في محركات البحث (أقصى 160 حرف)

### محتوى المقالة
- يمكنك إضافة عدة أقسام للمقالة
- لكل قسم:
  - **عنوان القسم**: عنوان فرعي
  - **المرساة (Anchor)**: معرّف فريد للقسم (مثال: section-one)
  - **محتوى القسم**: النص الأساسي
  - **ترتيب الظهور**: تحديد ترتيب الأقسام

### خيارات النشر
- **نشر المقالة الآن**: اختر هذا الخيار لنشر المقالة فوراً

## هيكل قاعدة البيانات

يتم حفظ المقالات مع المعلومات التالية:

```prisma
- id: معرّف البريمري
- categoryId: معرّف الفئة
- slug: نسخة من العنوان قابلة للاستخدام في الرابط
- title: عنوان المقالة
- excerpt: ملخص
- coverImageUrl: رابط صورة الغلاف
- publishedAt: تاريخ النشر
- status: حالة المقالة (مثل: published, draft)
- seoTitle: عنوان SEO
- seoDescription: وصف SEO
- sections: الأقسام المتعددة
- media: الصور والملفات المرفقة
```

## API Endpoints

### الحصول على المقالات
```
GET /api/admin/articles
```

### إنشاء مقالة جديدة
```
POST /api/admin/articles
Body: {
  title: string,
  excerpt: string,
  categoryId: number,
  coverImageUrl: string,
  seoTitle: string,
  seoDescription: string,
  sections: Array,
  publishNow: boolean
}
```

### الحصول على مقالة واحدة
```
GET /api/admin/articles/:id
```

### تحديث مقالة
```
PUT /api/admin/articles/:id
Body: { ...نفس حقول الإنشاء }
```

### حذف مقالة
```
DELETE /api/admin/articles/:id
```

### الحصول على الفئات
```
GET /api/admin/categories
```

## نصائح الاستخدام

1. **العنوان الفعال**: اختر عناوين جذابة وواضحة تصف محتوى المقالة
2. **الملخص**: اكتب ملخص جيد يشجع القارئ على فتح المقالة
3. **الصور**: استخدم صورة غلاف مناسبة ومرخّصة أو صورة محلية من المشروع
4. **SEO**: استخدم الكلمات المفتاحية المناسبة في العنوان والوصف
5. **الأقسام**: قسّم المحتوى لأقسام متعددة لسهولة القراءة
6. **المرساة**: إذا تُركت فارغة فسيتم توليد مرساة فريدة تلقائياً أثناء الحفظ
7. **المصادر**: أي أرقام أو نسب أو توصيات أو نتائج دراسات داخل المسودة يجب أن تكون من مصدر موثوق مع توثيق واضح داخل النص أو في بيانات المسودة
